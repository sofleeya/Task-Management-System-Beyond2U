## 💡 Introduction

The **Beyond2U Task Management System** is a web-based application developed to streamline the management of **Tech Refresh** and **Asset Replacement** tasks within a corporate environment.

Previously, task updates were shared through platforms like **WhatsApp**, which often caused:

* ❌ Missing information
* ❌ Delayed updates
* ❌ Lack of proper tracking

### 🎯 Project Goals

* Centralize all task records
* Enable real-time progress tracking
* Send automatic notifications to Team Leads
* Maintain complete submission & verification history

### 👥 Team Structure

* 👨‍💼 1 Project Manager
* 👨‍💻 5 System Engineers

### ⏱️ Duration

* 10 weeks

## 💻 Final Product

The system is a **fully functional web application** with role-based access:

### 👥 User Roles

#### 👨‍💼 Team Lead

* View all task requests
* Approve / reject submissions
* Search & filter tasks

#### 👨‍💻 Engineer

* View assigned tasks
* Update task status
* Upload proof of completion

#### 🛠️ Admin

* Manage users
* Monitor system
* Maintain data consistency

---

### ✨ Core Modules

* 🔐 User Authentication
* 📋 Task Creation & Assignment
* 📂 Proof Upload System
* ✅ Approval & Rejection Workflow
* 📊 Reporting System

---

### 🧩 Key Features

* 📌 **Manage Requests Page**

  * Displays all tasks in table format
  * Includes search & filter

* 📎 **Proof Upload**

  * Attach images/documents
  * Stored with task record

* 🔄 **Approval Workflow**

  * ✅ Approve → status updated
  * ❌ Reject → allow resubmission

* 🪟 **Popup Modal**

  * View full task details without leaving page

* 📊 **Reporting System**

  * Shows:

    * Completed tasks
    * Pending tasks
    * Rejected tasks

---

## 🔄 System Workflow

1. Team Lead assigns task
2. Engineer completes task
3. Engineer uploads proof
4. Team Lead reviews:

   * ✅ Approve OR
   * ❌ Reject
5. System updates status automatically
6. Admin monitors system
